# ReelDrop 🎬

Video downloader for Instagram, YouTube Shorts, TikTok, Twitter, Facebook and 1000+ sites.

## Setup

**Requirements:** Node.js, Python 3, yt-dlp

### 1. Install yt-dlp
```bash
pip install yt-dlp
```

### 2. Install Node dependencies
```bash
npm install
```

### 3. Run the server
```bash
node server.js
```

### 4. Open the app
Go to → http://localhost:3000

## Usage
1. Paste a video URL
2. Select quality (Best / 1080p / 720p / 480p / 360p)
3. Choose format (MP4 video or MP3 audio)
4. Hit Fetch → then Download

## Notes
- Files are streamed directly — no permanent storage on server
- For personal use only — respect content creators
